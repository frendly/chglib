Дорогие коллеги!

                                               Приглашаем Вас принять участие в семинаре
      Инновационный проект издательства Karger по продвижению российских авторов.

                                              Программа семинара:
      1. Предложения участникам проекта по опубликованию научных статей в журналах Karger
      2. Редакционные услуги Karger по оптимизации качества рукописей, представленных участниками проекта, с учетом требований журналов
      3. Условия публикации статей участников проекта в открытых и гибридных журналах Karger
      4. Услуги Karger по продвижению публикаций участников проекта в профессиональных сетях с целью увеличения просмотров и цитирования
      5. Программа Karger по обучению авторов современным процессам подготовки и подачи рукописи
