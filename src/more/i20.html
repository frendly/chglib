Web of Knowledge — объединённая платформа, которая связывает базы данных с различными рубрикаторами.
Благодаря данной платформе, можно:
  • исследовать ведущие мировые базы данных по цитированию
  • проводить поиск по пристатейным ссылкам
  • использовать аналитические инструменты в рамках единого интерфейса.

    Используется унифицированный словарь поисковых терминов для быстрого и удобного поиска.
