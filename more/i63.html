База данных Cambridge Structural Database

Издатель - The Cambridge Crystallographic Data Centre - (основан в 1965 г.) –
всемирный банк данных низкомолекулярных органических и металлоорганических соединений кристаллических структур.

Общая характеристика ресурса.

Ежегодно пополняемая база данных о кристаллическом строении органических, металлорганических и некоторых макромолекулярных соединений.

Данные о координатах атомов, угловых и линейных межатомных параметрах для более чем 600000 соединений,
определенные в результате рентгеновских и нейтронных дифракционных анализов.
Дополнены библиографическими и химическими сведениями, а также информацией о физических свойствах.

Комплекс программ для работы с этой базой данных, с любыми кристаллическими соединениями – для биологов, химиков и кристаллографов,
в том числе программа для расшифровки структур из порошковых данных, предсказания конформаций гибких молекул, предсказания мотивов упаковки
некоторых молекул, для молекулярного докинга.

Возможность пользователям самим писать скрипты для работы с базой данных.

Рекомендации по доступу.

Поиск кристаллических структур по названию, химической формуле, литературному источнику и другим критериям.

Условия доступа:           по IP-адресам БНЦ
Адрес для работы:         https://www.ccdc.cam.ac.uk/
