Уважаемые читатели,
предлагаем вашему вниманию тестовый доступ к электронно-библиотечной системе [**Лань**](http://e.lanbook.com/ "до 3 мая 2015 года").

- **Условия доступа:**
  - для всех читателей доступ неограничен. Осуществляется не только из стен Библиотеки, но также из любой точки мира, где есть Интернет.
  - Для тестирования ресурса с рабочего места или из дома, необходимо зарегистрироваться в Личном кабинете, находясь в этот момент в стенах Библиотеки.
  - В самой Библиотеке работа с ЭБС [ Лань ](http://e.lanbook.com/ "до 3 мая 2015 года") \- по IP-адресам без регистрации.
- **Адрес для работы:** [ http://e.lanbook.com/](http://e.lanbook.com/)
- Сроки тестирования: до 3 мая 2015 г.

Подробности – на сайте БНЦ, в разделе [тестовый доступ](/Restmp.html).
