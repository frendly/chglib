При работе из стен Библиотеки логин, пароль не требуются.
Вход для доступа: https://ebookcentral.proquest.com/lib/benran/home.action
Вас должны опознать как Библиотеку по естественным наукам (Library for Natural Sciences).

Далее через Browse Subjects перейти на страницу с предметными рубриками.
Выбрав необходимый раздел, открываем список книг. Полные тексты доступны в режиме PDF (значок зеленого цвета).
Книги доступны периода 2014 - 2016 годов издания.

С ресурсом можно работать с рабочих мест (вне библиотеки), используя логин и пароль.
За дополнительной информацией, пожалуйста, обратитесь к дежурному библиотекарю.

Условия доступа: по IP-адресам БНЦ.
Адрес для работы: https://ebookcentral.proquest.com/lib/benran/home.action
Сроки тестирования: до 28 апреля 2017 г.
