Доступ организован с компьютеров электронного читального зала ЦБ НЦЧ, на одном из компьютеров в библиотеках-филиалах ИПТМ, ИФТТ, ИСМАН, ФинЭПХФ.

            Электронно-библиотечной система Лань представляет собой платформу,
на которой размещены электронные версии книг издательства «Лань» и других ведущих издательств учебной литературы.
Также здесь Вы найдете электронные версии периодических изданий по естественным, техническим и гуманитарным наукам.

Издания собраны в коллекции, сгруппированные по основным областям знаний.

Среди основных сервисных возможностей ресурса можно выделить следующие:

  •         поиск по автору и названию;
  •         контекстный поиск;
  •         личный кабинет пользователя;
  •         возможность сохранения или распечатки 10% книги за одну сессию.
Условия доступа: для всех читателей доступ неограничен. Осуществляется не только из стен Библиотеки, но также из любой точки мира, где есть Интернет.
Для тестирования ресурса с рабочего места или из дома, необходимо зарегистрироваться в Личном кабинете, находясь в этот момент в стенах Библиотеки.
В самой Библиотеке работа с ЭБС Лань - по IP-адресам без регистрации.

Адрес для работы: http://e.lanbook.com/
